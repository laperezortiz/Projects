# Churn telecommunications company project.
## Problem.
The telecommunications operator Interconnect would like to be able to forecast its customer churn rate. If a user is found to be planning to leave, they will be offered promotional codes and special plan options.
## Process.
To meet Interconnect's requirements, it was necessary to do a full machine learning pipeline, with four different datasets, some EDA, feature engineering, dataset merging, correlation analysis, data splitting, data scaling, training and evaluation of various models,and manual and grid search model hyperparameter tuning.
## Results.
The selected model achieved an AUC-ROC of 0.8468 and an accuracy of 0.7332 on the test set.
